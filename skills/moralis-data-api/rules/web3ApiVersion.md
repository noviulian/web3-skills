# Get API version

Check the current version of the Moralis Web3 API.

## Method

GET

## Base URL

`https://deep-index.moralis.io/api/v2.2`

## Path

`/web3/version`

## Example (curl)

```bash
curl -X GET "https://deep-index.moralis.io/api/v2.2/web3/version" \
  -H "accept: application/json" \
  -H "X-API-Key: $MORALIS_API_KEY"
```
