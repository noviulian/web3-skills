# Get weights of endpoints

View the cost and rate limits for each Moralis API endpoint.

## Method

GET

## Base URL

`https://deep-index.moralis.io/api/v2.2`

## Path

`/info/endpointWeights`

## Example (curl)

```bash
curl -X GET "https://deep-index.moralis.io/api/v2.2/info/endpointWeights" \
  -H "accept: application/json" \
  -H "X-API-Key: $MORALIS_API_KEY"
```
