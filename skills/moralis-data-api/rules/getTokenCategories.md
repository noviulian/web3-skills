# Get ERC20 token categories

## Method

GET

## Base URL

`https://deep-index.moralis.io/api/v2.2`

## Path

`/tokens/categories`

## Example (curl)

```bash
curl -X GET "https://deep-index.moralis.io/api/v2.2/tokens/categories" \
  -H "accept: application/json" \
  -H "X-API-Key: $MORALIS_API_KEY"
```
